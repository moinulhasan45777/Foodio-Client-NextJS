# Foodio - Next.js Frontend

A simple food ordering frontend built with **Next.js 13+**, **TypeScript**, and **Tailwind CSS**.  
This project includes login/logout handling and pages for Home, Menu, My Orders, and Sign-In.

---

## Features

- Login/Logout using `localStorage` token
- Conditional rendering of Sign In button vs user menu
- Pages:
  - Home
  - Food Menu
  - My Orders
  - Sign-In
- Minimal Tailwind CSS styling
- No hydration or cascading render errors

---

## Setup & Run Instructions

### 1. Clone the Repository

```bash
git clone <repo-url>
cd <repo-folder>
```

### 2. Install Dependencies

npm install

### 3. Run the Development Server

npm run dev
